
### 概览

本项目支持下载中国省市区三级瓦片下载，提供高德以及天地图系列的瓦片下载服务。

- 项目基于`Electron`+`Vue3`+`leaflet`

- 执行`yarn build:win`命令，会构建`windows`环境下的`.exe`和`.msi`安装包
